# Aggregating Explanations from Multi Models

This repository contains the code used in the final chapters of my PhD thesis, addressing in part what was presented in the "Beyond single-model XAI: aggregating multi-model explanations for enhanced trustworthiness" (*Vascotto, Rodriguez, Bonaita, Bortolussi*) [paper](https://ceur-ws.org/Vol-4132/short38.pdf) and the following extensions.
